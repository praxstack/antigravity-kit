# Quản lý preview

Quản lý máy chủ phát triển cục bộ của bạn với quy trình `/preview`.

## Tổng quan

Lệnh `/preview` quản lý máy chủ phát triển cục bộ của bạn.
Khởi động, dừng, khởi động lại hoặc kiểm tra trạng thái bằng các lệnh đơn giản.

<FeatureGrid>
  <Feature title="start" description="Khởi động máy chủ" />
  <Feature title="stop" description="Tắt máy chủ" />
  <Feature title="restart" description="Tải lại" />
  <Feature title="check" description="Tình trạng" />
</FeatureGrid>

## Quy trình

<StepList>
  <Step number={1} title="Khởi động máy chủ">
    ```bash
    /preview start
    ```
  </Step>
  <Step number={2} title="Kiểm tra trạng thái">
    ```bash
    /preview
    ```
  </Step>
  <Step number={3} title="Dừng máy chủ">
    ```bash
    /preview stop
    ```
  </Step>
</StepList>
