# 预览管理

使用 `/preview` 工作流管理你的本地开发服务器。

## 概述

`/preview` 命令负责管理你的本地开发服务器。
用简单的命令即可启动、停止、重启或查看状态。

<FeatureGrid>
  <Feature title="start" description="启动服务器" />
  <Feature title="stop" description="关闭服务器" />
  <Feature title="restart" description="重新加载" />
  <Feature title="check" description="健康状况" />
</FeatureGrid>

## 流程

<StepList>
  <Step number={1} title="启动服务器">
    ```bash
    /preview start
    ```
  </Step>
  <Step number={2} title="查看状态">
    ```bash
    /preview
    ```
  </Step>
  <Step number={3} title="停止服务器">
    ```bash
    /preview stop
    ```
  </Step>
</StepList>
