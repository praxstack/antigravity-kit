# プレビューの管理

`/preview` ワークフローで、ローカル開発サーバーを管理しましょう。

## 概要

`/preview` コマンドはローカル開発サーバーを管理します。
シンプルなコマンドで、起動、停止、再起動、状態の確認ができます。

<FeatureGrid>
  <Feature title="start" description="サーバーを起動" />
  <Feature title="stop" description="サーバーを終了" />
  <Feature title="restart" description="再読み込み" />
  <Feature title="check" description="健康状態" />
</FeatureGrid>

## プロセス

<StepList>
  <Step number={1} title="サーバーを起動する">
    ```bash
    /preview start
    ```
  </Step>
  <Step number={2} title="状態を確認する">
    ```bash
    /preview
    ```
  </Step>
  <Step number={3} title="サーバーを停止する">
    ```bash
    /preview stop
    ```
  </Step>
</StepList>
