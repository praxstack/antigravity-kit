# Triển khai production

Phát hành tự tin với quy trình `/deploy` đặt an toàn lên hàng đầu.

## Tổng quan

Lệnh `/deploy` là một **Nhạc trưởng Kiểm toán & Phát hành**.
Nó buộc code của bạn phải vượt qua một kiểm tra sức khỏe nghiêm ngặt trước khi triển khai.

<Callout type="info" title="Nó ngăn bạn triển khai nếu:">
  - Phát hiện lỗ hổng bảo mật
  - Tồn tại lỗi linting
  - Kiểu TypeScript không hợp lệ
  - Test quan trọng thất bại
</Callout>

## Mẹo hay

<ProTips>
  <Tip title="Kiểm tra trước">
    Chạy `/deploy check` để chạy thử các xác thực mà không triển khai.
  </Tip>
  <Tip title="An toàn rollback">
    Dùng `/deploy rollback` để lập tức quay về phiên bản ổn định gần nhất.
  </Tip>
</ProTips>
