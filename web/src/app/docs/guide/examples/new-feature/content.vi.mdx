# Thêm tính năng mới

Tìm hiểu cách thêm các tính năng phức tạp vào codebase có sẵn một cách an toàn với quy trình `/enhance`.

## Tổng quan

Quy trình `/enhance` là công cụ hằng ngày cho việc phát triển lặp.
Khác với `/create` (dựng từ đầu), `/enhance` tập trung thêm chức năng một cách phẫu thuật mà không phá vỡ code hiện có.

<Callout type="info">
  **Tình huống:** Thêm tính năng "Dashboard Analytics" vào một ứng dụng SaaS có sẵn.
</Callout>

## Quy trình

<StepList>
  <Step number={1} title="Lập kế hoạch tính năng">
    Mô tả yêu cầu tính năng của bạn thật rõ ràng.

    ```bash
    /enhance Add an analytics dashboard for user activity
    Requirements:
    - Track daily active users and page views
    - Visual charts using Recharts
    - API endpoint to aggregate data
    ```
  </Step>
  <Step number={2} title="Triển khai thành phần">
    Các chuyên gia triển khai trọn vẹn lát cắt dọc (vertical slice).

    <FeatureGrid cols={4}>
      <Feature title="Database" />
      <Feature title="Backend" />
      <Feature title="Frontend" />
      <Feature title="Testing" />
    </FeatureGrid>
  </Step>
  <Step number={3} title="Kiểm tra tích hợp">
    Chạy `final checks` để xác thực kích thước bundle, khả năng truy cập và kiểm thử.
  </Step>
</StepList>

## Phiên làm việc ví dụ

<Terminal>

  </TerminalBlock>
