# プロジェクトの状態

`/status` ワークフローで、プロジェクトの全体像を把握しましょう。

## 概要

`/status` コマンドは、プロジェクトの**完全なダッシュボードビュー**を提供します。
プロジェクト情報、技術スタック、完了した機能、エージェントの活動、プレビューの状態を確認できます。

## プロセス

<StepList>
  <Step number={1} title="コマンドを実行する">
    コマンドを実行するだけで、プロジェクトのダッシュボードが表示されます。

    ```bash
    /status
    ```
  </Step>
</StepList>
