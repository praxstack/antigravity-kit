# Trạng thái dự án

Có cái nhìn tổng quan đầy đủ về dự án của bạn với quy trình `/status`.

## Tổng quan

Lệnh `/status` cho bạn một **góc nhìn dashboard đầy đủ** về dự án.
Xem thông tin dự án, tech stack, các tính năng đã xong, hoạt động của tác tử và trạng thái preview.
