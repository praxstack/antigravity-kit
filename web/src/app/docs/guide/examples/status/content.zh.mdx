# 项目状态

使用 `/status` 工作流，全面掌握你项目的整体情况。

## 概述

`/status` 命令为你提供项目的**完整仪表盘视图**。
查看项目信息、技术栈、已完成的功能、智能体活动以及预览状态。

## 流程

<StepList>
  <Step number={1} title="运行命令">
    只需运行命令即可查看你的项目仪表盘。

    ```bash
    /status
    ```
  </Step>
</StepList>
