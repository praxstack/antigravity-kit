# テストの生成と実行

`/test` ワークフローで、テストを生成して実行しましょう。

## 概要

`/test` コマンドは**テストの生成と実行**を担います。
コードを分析し、テストスイートを作成し、テストを実行したり、カバレッジを確認したりします。

<FeatureGrid>
  <Feature title="run" description="テストを実行" />
  <Feature title="generate" description="テストを作成" />
  <Feature title="coverage" description="レポート" />
  <Feature title="watch" description="ライブモード" />
</FeatureGrid>

## プロセス

<StepList>
  <Step number={1} title="すべてのテストを実行する">
    ```bash
    /test
    ```
  </Step>
  <Step number={2} title="ファイルのテストを生成する">
    ```bash
    /test src/auth.ts
    ```
  </Step>
  <Step number={3} title="カバレッジを確認する">
    ```bash
    /test coverage
    ```
  </Step>
</StepList>
