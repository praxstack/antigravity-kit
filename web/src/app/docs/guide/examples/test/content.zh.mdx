# 测试生成与执行

使用 `/test` 工作流生成并运行测试。

## 概述

`/test` 命令负责**测试的生成与执行**。
分析代码、创建测试套件、运行测试或检查覆盖率。

<FeatureGrid>
  <Feature title="run" description="执行测试" />
  <Feature title="generate" description="创建测试" />
  <Feature title="coverage" description="报告" />
  <Feature title="watch" description="实时模式" />
</FeatureGrid>

## 流程

<StepList>
  <Step number={1} title="运行所有测试">
    ```bash
    /test
    ```
  </Step>
  <Step number={2} title="为文件生成测试">
    ```bash
    /test src/auth.ts
    ```
  </Step>
  <Step number={3} title="检查覆盖率">
    ```bash
    /test coverage
    ```
  </Step>
</StepList>

## 实用技巧

<ProTips>
  <Tip title="测试行为">
    关注函数做什么，而非它如何实现。
  </Tip>
  <Tip title="边界情况">
    始终测试错误、空输入和各种边界。
  </Tip>
</ProTips>
