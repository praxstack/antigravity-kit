# Sinh và chạy kiểm thử

Sinh và chạy test với quy trình `/test`.

## Tổng quan

Lệnh `/test` lo việc **sinh và chạy kiểm thử**.
Phân tích code, tạo bộ test, chạy test hoặc kiểm tra độ phủ.

<FeatureGrid>
  <Feature title="run" description="Chạy test" />
  <Feature title="generate" description="Tạo test" />
  <Feature title="coverage" description="Báo cáo" />
  <Feature title="watch" description="Chế độ trực tiếp" />
</FeatureGrid>

## Quy trình

<StepList>
  <Step number={1} title="Chạy toàn bộ test">
    ```bash
    /test
    ```
  </Step>
  <Step number={2} title="Sinh test cho file">
    ```bash
    /test src/auth.ts
    ```
  </Step>
  <Step number={3} title="Kiểm tra độ phủ">
    ```bash
    /test coverage
    ```
  </Step>
</StepList>
