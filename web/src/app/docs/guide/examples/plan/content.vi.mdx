# Lập kế hoạch dự án

Tạo các kế hoạch triển khai chi tiết trước khi viết code với quy trình `/plan`.

## Tổng quan

Lệnh `/plan` kích hoạt **chế độ chỉ lập kế hoạch**.
Nó dùng tác tử `project-planner` để tạo một file kế hoạch chi tiết mà không viết bất kỳ dòng code nào.

<Callout type="info">
  **Quy tắc then chốt:** KHÔNG viết CODE trong lúc lập kế hoạch. Chỉ tạo file kế hoạch `{task-slug}.md` (ở thư mục gốc dự án).
</Callout>

## Quy trình

<StepList>
  <Step number={1} title="Mô tả dự án của bạn">
    Cho planner biết bạn muốn xây gì.

    ```bash
    /plan e-commerce site with cart and checkout
    ```
  </Step>
  <Step number={2} title="Câu hỏi Socratic">
    Planner đặt các câu hỏi làm rõ trước khi tạo kế hoạch.
  </Step>
  <Step number={3} title="Kế hoạch được tạo">
    Tạo `{task-slug}.md` ở thư mục gốc dự án kèm các tác vụ và phân công tác tử.
  </Step>
</StepList>

## Mẹo hay

## Bước tiếp theo

- Chạy `/plan` cho tính năng lớn tiếp theo của bạn
- Sau khi lập kế hoạch, dùng [Tạo ứng dụng](/docs/guide/examples/create) để triển khai
