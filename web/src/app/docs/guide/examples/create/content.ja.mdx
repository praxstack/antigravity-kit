# 新しいアプリの作成

`/create` ウィザードで、完全なアプリケーションをゼロから構築しましょう。

## 概要

`/create` コマンドはあなたの**アプリケーション作成ウィザード**です。
対話形式のやり取りを通じて案内し、要件を理解し、
アーキテクチャを計画して、各専門エージェントを編成しアプリを構築します。

<Callout type="info">
  **アイデアから動くアプリへ：** 作りたいものを説明するだけで、ウィザードが技術スタック、ファイル構成、データベーススキーマ、実装まで面倒を見ます。
</Callout>

## プロセス

<StepList>
  <Step number={1} title="アプリを説明する">
    何を作りたいかをウィザードに伝えます。必要に応じて確認の質問をします。

    ```bash
    /create e-commerce site with product listing and cart
    ```
  </Step>
  <Step number={2} title="計画を確認する">
    `project-planner` エージェントが詳細な計画を作成します。承認すると先へ進みます。
  </Step>
  <Step number={3} title="エージェントが構築する">
    専門エージェントが並行して作業します：database-architect、backend、frontend。
  </Step>
</StepList>
