# 创建新应用

使用 `/create` 向导从零构建完整的应用。

## 概述

`/create` 命令是你的**应用创建向导**。
它会通过一段交互式对话引导你，理解你的需求，
规划架构，并编排各个专家智能体来构建你的应用。

<Callout type="info">
  **从想法到可运行的应用：** 只需描述你想要什么，向导会负责技术栈、文件结构、数据库模式和具体实现。
</Callout>

## 流程

<StepList>
  <Step number={1} title="描述你的应用">
    告诉向导你想构建什么。如有需要，它会提出澄清性的问题。

    ```bash
    /create e-commerce site with product listing and cart
    ```
  </Step>
  <Step number={2} title="审阅计划">
    `project-planner` 智能体会生成一份详细的计划。批准后即可继续。
  </Step>
  <Step number={3} title="智能体开始构建">
    专家智能体并行工作：database-architect、backend、frontend。
  </Step>
</StepList>

## 实用技巧
